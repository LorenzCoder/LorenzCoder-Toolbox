# 🛠️ Lorenz-Toolbox

Willkommen in meiner Open-Source-Schmiede! Hier sammle ich Skripte, Listen und kleine Tools, die meinen Windows-Alltag erleichtern und automatisieren.

Als **Open-Source-Mensch** teile ich diese Tools, damit andere sie forken, verbessern oder einfach nutzen können.

---

## 📂 Kategorien

Hier findest du die verschiedenen Bereiche meiner Toolbox:

- **[📁 Ordnerzeug](./Ordnerzeug/)**
  Skripte zur Dateiverwaltung, zum Aufräumen von Verzeichnissen und zum Sortieren von Downloads.
- **[🏗️ Winget-Setup (Coming Soon)](#)**
  Listen und Skripte, um ein frisches Windows-System in Minuten mit allen wichtigen Programmen aufzusetzen.

---

## 🚀 Installation & Nutzung

1.  Repository clonen:
    ```powershell
      git clone git clone https://github.com/LorenzCoder/LorenzCoder-Toolbox.git
    ```
2.  In das entsprechende Verzeichnis wechseln und die Dokumentation in der dortigen `README.md` lesen.
3.  _Hinweis:_ Eventuell musst du die Execution Policy in der PowerShell anpassen (`Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`).
